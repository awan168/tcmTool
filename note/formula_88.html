==============================  
	


洗肝明目湯


【出 典】
     

萬病回春

【組 成】
     

當　歸 1.0　　赤　芍 1.0　　黃　芩 1.0　　連　翹 1.0　　川　芎 1.0

生地黃 1.0　　山梔子 1.0　　防　風 1.0　　荊　芥 1.0　　羌　活 1.0

決明草 1.0　　薄　荷 1.0　　蔓荊子 1.0　　杭　菊 1.0　　黃　連 1.0

蒺　藜 1.0　　橘　梗 1.0　　甘　草 1.0　　石　膏 1.0

【效 能】
     

清熱袪風，消腫明目。

【主 治】
     

治一切肝經風熱所致的目赤腫痛。

【入 經】
     

足厥陰陽明

【應 用】
     

宜熱症眼疾、炎症、充血、疼痛、硬化性角膜炎、角膜實質炎、鞏膜炎、虹彩炎、綠內障。

【說 明】
     

本方主要用於實證的眼疾，而兼有炎症充血、疼痛刺激症狀，對角膜及結膜疾患廣被應用。

方中當歸、芍藥、川芎、地黃即四物湯，能增血明目；黃連、黃岑、梔子均為苦寒之品，能清解實熱；羌活、防風、桔梗、荊芥、薄荷皆兼有發散與解毒排毒之功能；決明子入肝經除風熱，治一初目疾；石膏清熱降火；蔓荊子入肝經，涼血洏治目赤；菊花平肝養目，去翳膜；蒺藜散肝經風熱、明目；甘草調和諸藥以增療效。

【加 減】
     

1．風熱腫痛：加升麻、葛根。

2．風熱流淚：加夏枯草、香附。

3．風熱障翳：細辛、密蒙花。

4．風毒暴作赤腫：加大黃。

【用 法】
     

飯前或飯後溫服。
